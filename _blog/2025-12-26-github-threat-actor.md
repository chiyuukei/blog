やあ、僕の短い日記へようこそ。ここでは、どうやって「Threat Actor（脅威アクター）」が「GitHub」のようなプラットフォームを利用して「ソーシャルエンジニアリング」攻撃を仕掛けるのかについて話すよ。公式な記事ではなく、暇なときに書いた日記だから、伝え方に間違いがあったらごめんね。正直ちょっと退屈かもしれないし。じゃあ、さっそくいこう。

ところで、こういう手口は色んなプラットフォームでよく見つかるんだけど、そのひとつがGitHubだ。ここはGit由来のコードを保存できる場所を提供しているだけじゃなく、ユーザーが無料でWebホスティングもできる。で、たいていそれが悪意あるアクターの侵入口になったりするんだ。

さて、これは短い日記だから長々とは話さないよ。あ、そうだ、その前に言っとくと、これは偶然見つけたんだ、へへ。つまり、暇つぶしってわけで、もちろんお金はもらってない。最初からあてもなく冒険してたからね。

まず最初に、リアルタイムのフィッシングURLの一覧を探して、ここで見つけた：
<a href="https://openphish.com">openphish.com</a>。このプラットフォームは無料のフィッシングリンクをまとめて提供してくれる（有料プランもあるけど、無料一択。だってお金払うの嫌いだし、へへ）。で、もうターゲットが見つかったことにしよう。これがそのGitHubアカウント：

```
hxxps://github[.]com/veqqzy
```

次のステップでは、そのアカウントのリポジトリを調べた。ついでに証拠としてサンプルをダウンロードしてアーカイブもしたよ。なんでダウンロードするのかって？ そりゃ証拠のためだよ。それにWebhookも入ってたから、ちょっとした悪さもできるしね、へへ（  ≧ᗜ≦)

三つ目のステップは、ダウンロード／アーカイブした後に、そのアクターのアカウント履歴を調べること。これは大好きな作業。趣味みたいなもんだから。アカウントの履歴を調べるのにはGoogleレンズを使ったよ。プロフィール画像が、昔観たアニメのキャラだったんだけど、タイトルを忘れててね。もしかしてオタクかなーって。ちなみにURLはこれ：

```
hxxps://avatars[.]githubusercontent[.]com/u/253676428?v=4
```

そのプロフィール画像を逆引き検索したら、そのキャラ名が「Licht Bach」だって分かった。アニメの内容はうろ覚えだけど、筋はまだわりと覚えてる。

四つ目のステップとして、ターゲットのプロフィール調査はやったけど、画像のメタデータみたいな成果は出なかった。でも、アクターが残していった超重要なものがある。何かって？ そう、メールアドレス。なんで重要かって？ GitHubクラスのプラットフォームでも登録にはメールが必要だからね。僕もそうだし。ただ、プライベートメールを非表示にする設定もある。でも残念ながら今回は何も見つからなかった（空振り）。だから手元にある手がかりは、その人のコードに埋め込まれていたWebhookだけ。まあいいや。この日記の目的は、あのアクターの動機やトレンドをさらっと明かすことだから、深追いはしない。

悩むくらいなら、コードに仕込まれているWebhookのAPIにスパム／削除でも投げとくかって決めた。たぶんDiscordのWebhookだと思う。Discordは、アカウント作ってサーバー運営する人向けに無料でBotを作れるし、アクターの多くはそれをC2として使ってマルウェアやフィッシングページの通信に使ってたりする。だからもう、ちょっと言葉攻めでもしてやろうかな、へへ…

あ、やっぱそれは最後にしよう。これはある意味、傑作だよ。GitHub上で堂々とこういうことやってるやつがいるし、しかもJavaScriptのコードが難読化されてない。だから僕には面白かった。アクターの性別を分析した結果、彼は男だと推測。というのも、下のリポジトリで、異性に向けたっぽいメッセージを書いていたから：

```
hxxps://github[.]com/veqqzy/love-massage
```

以上、僕の短い日記でした。これはプロのハンティングとは言えないね。コードの解析結果や目的（リバースエンジニアリング）を書いてないし、僕はどちらかというと観察者でいる方が好き。こうしてメモしておく、みたいな感じで。

最後のメッセージはシンプル。こういうのには気をつけて、常に用心してね。悪意のあるアクターの脅威は現実に存在するし、時間が経つほど手口は洗練され、常にアップデートされていくから。
